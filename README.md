Step Around Bogor - Virtual Tour Guide AR
Aplikasi Step Around Bogor adalah aplikasi web berbasis Augmented Reality (AR) yang memungkinkan pengguna untuk melakukan scanning marker logo destinasi di Kebun Raya Bogor. Saat marker terdeteksi, aplikasi akan menampilkan objek 3D terkait dan panel informasi lengkap tentang destinasi tersebut.
